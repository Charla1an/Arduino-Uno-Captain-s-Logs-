# Arduino Uno (Captain’s Logs)

This repository documents my journey learning Arduino through hands-on projects, experiments, and incremental improvements.

I started with little to no experience in electronics or embedded systems, and this log captures how I am building that understanding step by step by actually wiring circuits, writing code, and debugging real problems.

Rather than focusing on polished final results, this repository emphasizes:

- learning through building
- understanding how and why things work
- improving logic and structure over time

## What you'll find here

- Small projects and experiments (LED control, sensors, basic systems)
- Progression from simple implementations to more structured solutions
- Notes on mistakes, fixes, and what I learned from each build

## Goal

To develop a solid foundation in Arduino, electronics, and embedded thinking by consistently building and improving real systems.
